# wamp-scala?
[wamp](http://wamp-proto.org/)의 scala 구현체이다.

# Contribution
개발에 참여하기 위한 가이드

## Development Enviornment
* Language: Scala(2.11)
* Framework: [Akka](http://akka.io)(2.4.1)
* Build: [sbt](http://www.scala-sbt.org/)(0.13.8)


## Package
```bash
> ./sbt package packageSrc 
```

## Test
```bash
> ./sbt test
```
